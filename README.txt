Welcome to the Virtual Coin Flipping Program

This program will simulate a coin flip using a random number generator.
Numbers 1-50 will come up as "Heads" and numbers 51-100 will come up as "Tails".

To compile the program use g++ via commandline.

Example:
	To Compile -> "g++ main.cpp coin.cpp -o coin"
	To Execute Program -> "./coin"
